ip2country
==========
|travis|_ |coveralls|_ |pypibadge|_

-----
About
-----
Lookup country code (iso 3166) by IP address.

-------
Install
-------

.. code:: shell

  $ pip install ip2country

Or

.. code:: shell

  $ python setup.py install

-------------
Run the tests
-------------

.. code:: shell

  $ tox
